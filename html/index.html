Image Search Abstraction Layer

User Stories

1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.

2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.

3. I can get a list of the most recently submitted search strings.

How to Use

    Pass a search term by adding the following:

    /api/imagesearch/<search term>

    View a specific page of the results by passing an offset:

    /api/imagesearch/<search term>?offset=2

    To view the most recent 10 searches, add the following parameters to the end of the URL

    /api/latest/imagesearch/

Example URL

    http://image-search-gl.herokuapp.com/api/imagesearch/puppies

    http://image-search-gl.herokuapp.com/api/imagesearch/funny kittens?offset=2
